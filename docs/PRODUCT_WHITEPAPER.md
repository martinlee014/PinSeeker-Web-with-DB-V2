
# PinSeeker Web - 产品白皮书 & 操作流程 (SOP)

## 1. 产品定位
**PinSeeker** 是一款基于 Web 的高精度高尔夫策略与分析工具。它不仅仅是一个测距仪，而是球手的**智能电子球童 (Smart Electronic Caddie)**。

### 核心价值主张
*   **可视化风险管理**：通过散布圆 (Dispersion Circles) 帮助球手理解击球落点概率，避开障碍区。
*   **私有化地图数据**：解决偏远/私人球场无数据痛点，提供专业级地图编辑器。
*   **云端同步体验**：多设备无缝切换，数据永不丢失。
*   **零门槛社交**：通过 QR 码一键邀请好友入局，无需繁琐注册流程。

## 2. 目标受众 (Target Audience)

| 用户类型 | 特征描述 | 核心需求 |
| :--- | :--- | :--- |
| **策略型球手** | 差点 10-20，注重成绩管理 | 不再盲目进攻，需要知道“哪里是安全的失误区域”。 |
| **赛事组织者** | 经常组织四人组或小比赛 | 需要快速让朋友加入比赛，并可能需要帮没有 APP 的朋友记分。 |
| **私享家/会员** | 常打非主流或私人球场 | 主流 App (18Birdies/TheGrint) 没有他们球场的地图，或者地图不仅更新。 |
| **教练/教学机构** | 需要复盘工具 | 需要在地图上画线、标记，向学员解释球场策略。 |

## 3. 核心功能亮点

### A. 击球散布分析 (Dispersion Analysis) 🔥
不同于传统 APP 只显示一个“点”，PinSeeker 根据用户设置的 **Carry (飞行距离)**、**Side Error (左右偏差)** 和 **Depth Error (纵深偏差)**，在地图上渲染出一个蓝色的概率椭圆。
> **价值**：直观展示90%的落点范围。如果椭圆覆盖了水障碍，球手就知道必须调整瞄准方向，直到椭圆完全处于安全区。

### B. 无缝赛事与社交 (Seamless Social Play) 🏆
极速组局体验。
> **QR 邀请**：在比赛大厅生成二维码，队友扫码即连（Auto-Join），无需手动输入六位代码。
> **Guest 模式**：队友没带手机？房主可添加 "Guest Player"，在同一台设备上为多人记分。
> **多人记分卡**：全新的记分界面支持 Tab 切换，一键提交全组（Me + Guest）成绩，实时同步云端。

### C. 云生态系统 (Cloud Ecosystem) ☁️
支持多用户登录与数据同步。
> **价值**：用户的球杆数据（Bag Data）和历史战绩（Round History）自动保存至云端。支持单设备互斥登录（Single-Device Policy），保障数据一致性。
> **HDCP 建模**：新用户只需输入差点 (Handicap)，系统自动生成全套球杆数据模型，降低配置门槛。

### D. DIY 球场编辑器 (Course Editor) 🛠️
内置专业级 GIS 地图编辑器，支持 22 级深度变焦。
> **价值**：用户可以拖拽 Tee 台和果岭位置，自定义 Par 信息。真正做到“我的主场我做主”。

### E. 交互式策略板 (Interactive Strategy) ✏️
提供画笔、文字备注、旗帜标记工具，以及可视化的风向罗盘。
> **价值**：像职业比赛的码数本一样做笔记，标记果岭速度、隐蔽沙坑等。

## 4. 竞品差异化对比

| 功能维度 | PinSeeker (本项目) | 主流竞品 (18Birdies/Arccos) |
| :--- | :--- | :--- |
| **测距逻辑** | **散布概率模型 (由此决定瞄准点)** | 简单的点对点直线距离 |
| **组局体验** | **QR 扫码 + Guest 代理记分** | 需全员下载 APP 并注册 |
| **地图精度** | **用户可编辑 (Level 22 极清)** | 官方数据库 (更新慢，通用精度) |
| **数据存储** | **云端同步 + 跨设备 + 冲突检测** | 本地存储或付费云服务 |
| **使用门槛** | **Web App (即开即用)** | 需要下载几百 MB 安装包 |

## 5. 下场全流程 SOP (User Journey)

### 第一阶段：登录与准备 (Login & Prep)
1.  **身份识别**: 新用户输入 ID 后，直接设定 HDCP，系统自动配置球包。
2.  **数据检查**: 登录时自动拉取云端数据，若有冲突提供可视化解决选项。

### 第二阶段：组局 (The Setup)
1.  **创建比赛**: 点击 `Host Tournament`。
2.  **邀请**: 朋友扫描屏幕上的 QR 码加入。
3.  **添加路人**: 对于没有 App 的球友，点击 `Add Guest` 将其加入列表。

### 第三阶段：发球台 (The Tee Box)
1.  **选择球手**: 点击 `Play Round`，选择 **"Me"** 或 **"Guest"** 视角。
2.  **策略制定**: 观察 Driver 落点散布圈。如果右侧有水，拖动瞄准滑块向左，直至蓝圈避开障碍。
3.  **击球**: 按照 APP 指示方向击球。

### 第四阶段：球道行进 (Fairway)
1.  **记录位置**: 到达球位后，点击底部 **GPS 按钮**。
2.  **攻果岭**:
    *   开启 **测距模式 (Ruler)**，点击 "MY LOC" 确认当前到旗杆距离。
    *   若果岭硬，使用 **画笔 (Pen)** 标记备注。

### 第五阶段：果岭结算 (Green)
1.  **多人记分**: 点击右上角旗帜图标，弹出记分板。
2.  **Tab 切换**: 在顶部切换 "Me", "Tom", "Jerry"。
3.  **保存**: 输入推杆数后点击 `Save All Scores`，数据一次性上传。

### 第六阶段：赛后复盘 (Post-Round)
1.  **结束**: 点击 `Finish Round` 查看记分卡。
2.  **回放**: 进入 **Replay Mode**，查看每一杆的实际轨迹（实线）与计划散布（虚线）。
